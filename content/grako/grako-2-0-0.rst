Grako 2.0.0
===========

:date: 2013-05-19
:slug: grako-2-0-0
:author: Apalala

.. :tags:
.. :category:
.. :summary:


|grammar_diagram|

Grako 2.0.0 was just released. This new version includes many
simplifications to the generated grammars, semantics through delegation,
a complete \ *antlr2grako* example, and the ability to create silly
diagrams of the grammars through *graphviz*. For a full list of changes,
see:

`PyPy <https://pypi.python.org/pypi?:action=display&name=grako&version=2.0.0>`__ or
the `Bitbucket
changelog <https://bitbucket.org/apalala/grako/commits/all>`__.
