Grako 1.0.0
===========

:date: 2013-02-12
:slug: grako-1-0-0
:author: Apalala
:tags: grako, peg, parser-generator, python, release




`Grako <https://bitbucket.org/apalala/grako>`__ 1.0.0 has been
released to `PyPi <http://pypi.python.org/pypi/grako/1.0.0>`__. It's
feature complete, and very fast.

I'm currently using it to parse and translate millions of lines of
`COBOL <http://en.wikipedia.org/wiki/COBOL>`__.

Enjoy!
